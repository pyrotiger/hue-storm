# hue-storm
* Script to manipulate office Hue lights
* Depends on [OpenHue](https://github.com/openhue)

# Logic Bomb
* What it really is a logic bomb in disguise for the Hue bridge. 
* USE AT YOUR OWN RISK!

😈

* My original intent was to create a "magneta" storm effect, unfortunately...
* Due to the limitations of these particular Hue devices, I inadvertently created
* a logic light bomb for the office!
